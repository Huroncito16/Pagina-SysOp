Temario de la presentación

¿Qué es un sistema operativo?

Un sistema operativo es el software principal que administra el hardware de la computadora y brinda servicios a los programas. Controla recursos como la memoria, el procesador, los dispositivos de entrada/salida y permite la interacción entre el usuario y la máquina.

Procesadores

El procesador o CPU es la “unidad central de procesamiento” de la computadora. Ejecuta instrucciones y coordina todas las operaciones del sistema. Existen arquitecturas de uno o varios núcleos, que permiten procesar más tareas en paralelo y aumentar el rendimiento.

Arquitectura de computadoras (Harvard y von Neumann)

La arquitectura de von Neumann utiliza una sola memoria para instrucciones y datos, lo que simplifica el diseño pero puede causar “cuellos de botella”. La arquitectura Harvard, en cambio, separa memoria de datos y de instrucciones, permitiendo mayor velocidad y eficiencia en algunos casos.

Hardware y software

El hardware es la parte física del computador: procesador, memoria, disco duro, periféricos. El software son los programas e instrucciones que le indican al hardware qué hacer. Ambos trabajan en conjunto: el hardware sin software no tiene utilidad, y el software necesita del hardware para ejecutarse.

Figuras importantes del software libre

Entre los pioneros del software libre se destacan Richard Stallman, fundador del proyecto GNU y la Free Software Foundation; Linus Torvalds, creador del kernel de Linux; y otros como Eric Raymond y Bruce Perens, que ayudaron a popularizar el movimiento de código abierto.

Instalación y uso

La instalación de un sistema operativo implica preparar el disco duro, copiar los archivos base del sistema y configurar parámetros iniciales. En el uso diario, el sistema operativo permite abrir programas, gestionar archivos, instalar controladores y garantizar la seguridad del equipo.

Arquitectura de computadoras

La arquitectura de computadoras describe la organización interna del hardware: CPU, memoria principal, buses de comunicación y dispositivos de entrada/salida. Su objetivo es lograr un sistema eficiente, rápido y que aproveche al máximo los recursos disponibles.
